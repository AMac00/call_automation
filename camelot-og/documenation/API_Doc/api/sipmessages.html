SipMsgObject API¶

SipMsgObject object contains a single SIP message and it’s headers.

API Methods¶

class camelot.vapi.vapi_endpoint_query_control.SipMsgObject(conn, callref, msgtype, methodname, respcode, msgid)[source]¶

SipMsgObject object contains single SIP message and it’s headers.

With the SipMsgObject user can retrieve the content bodies associated with the SIP message. User can get full sip message headers or selective headers using this object.
