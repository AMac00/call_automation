Camelot Logger API¶

The Camelot Logger API provides access to the Camelot module logging.

API Methods¶

Created on 17-Feb-2014

@author: smaturi

camelot.camlogger.disable_logging()[source]¶

Disables Camelot module style logging.

camelot.camlogger.enable_logging()[source]¶

Enables Camelot module style logging.

camelot.camlogger.getLogger(name)[source]¶

Creates logger in the Camelot module

Parameters:name – name for the logger
camelot.camlogger.setLevel(level)[source]¶

Set’s logger level for Camelot logging

Parameters:value – logging level
